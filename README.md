## How To Use üîß

Clone or download zip, install dependencies, and use by running commands using npm:

```bash
# Install dependencies
$ npm install

# Start development server
$ npm start

# Production build
$ npm build
# Development build
$ npm dev
```

Development server: `http://localhost:3000/`
